# 关于本站

本站是大概是博主的第三（还是第四？）个博客。

博主最近（2025-08）新入坑了方舟，并很喜欢白面鸮，所以预计在不久的将来这网站会有更多白面鸮或者莱茵生命的元素。

## 使用的技术/服务

网站部署/自动化流程：

- [GitHub Pages](https://pages.github.com/) (via [GitHub Actions](https://github.com/features/actions))
  - 网站自动化部署
- [CloudFlare](https://cloudflare.com/)
  - 全球 CDN 访问加速
- [AWS CloudFront](https://aws.amazon.com/cloudfront/)
  - 全球（尤其是国内）网络加速
- [CloudFlare R2](https://www.cloudflare.com/developer-platform/products/r2/)
  - 文章内图片等资源的存储

访客统计分析：

- [Google Analytics](https://developers.google.com/analytics)
  - 主要的访问指标（metrics）收集
- [Micorost Clarity](https://clarity.microsoft.com/)
  - 用户行为分析，虽然我也不知道有何意味
